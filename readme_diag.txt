how to use:

1. move the dialogue.json into your song's data folder
2. edit the json:
	img: background image
	text: what the character should say
	char: the character folder name
3. open the toolbox and go to the "Song Config" tab, select the song you want to have dialogue and type the path of the dialogue.hx code into the cutscene and/or end cutscene box (ex. "cutscenes/dialogue.hx"
4. open the song in story mode and it should work
